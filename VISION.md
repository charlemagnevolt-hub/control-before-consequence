# Control Before Consequence

## North Star

AI systems increasingly possess the ability to cause external consequences.

The central challenge is no longer whether an AI system can act.

The central challenge is whether a proposed action should be allowed to become consequential.

This project explores runtime admissibility as a control framework for agentic systems.

```text
Control Before Consequence
        ↓
Runtime Admissibility
        ↓
Admissibility Gate
        ↓
Reference Implementations
        ↓
Demonstrators
```
